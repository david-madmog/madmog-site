Super Terraria World

Super Terraria World changes the hit game Terraria into an MMORPG. It is a standalone roleplaying mod with fully fledged quests that will be kept up to date with the latest release of Terraria. I am the developer of the installer and have started helping port the mod to 1.3.5.



Fully Developed Questlines

Super Terraria World features fully developed questlines with custom items, custom areas and custom NPCs. Quests can contain everything from puzzles to crafting to combat.

Custom Locations

STW contains custom maps built by the talented builder Khaios which create an amazing atmosphere to run around in, completing quests. There are also areas allowing for the player to build, unleashing their creativity on the world!

Unique Items

STW boasts hundreds of new items including quest items, new weapons, new armour, new items and much more! All sprited by the talented pixel artist Jestex.

Massively Multiplayer

(When it is released) STW's Multiplayer will be in an MMORPG format - play with your friends in parties or solo, it is up to you. Thanks to all our Patreons for helping to pay for the servers!

Support on Patreon

Super Terraria World is kept running by the generous donators on our Patreon page - 100% of donations go towards the mod, and they are essential in order to keep servers running - not to mention the custom in-game rewards for donators!

A Terraria Backup Utility

T-Backup

T-Backup is a program that backs up worlds and player files from the game Terraria. It supports multiple profiles to allow different locations for mods. It is written in C#, and is intended to be run on Windows.
T-Backup does not require an installer, simply run the downloaded executable to start.
A new, improved version is currently in the works!


  • Main menu - most options and most utilities on this window.

  • Profile window - enables T-Backup to be used with mods.

  • Restore window - restore backups back into Terraria files.


Source code and downloads on GitHub!
Check the Terraria Forums thread for more info.

A Custom Terraria Logo Maker

Terraria Logo Maker

Terraria logo maker is a tool that allows you to create custom logos like the one from the game Terraria. It has support for custom themes, but only includes 'Overworld' at the moment. Once more themes have been made, they will be added as optional additional downloads. If you wish you can create your own theme using this tutorial.
Terraria Logo Maker is written in C# and intended to be run on Windows. Simply extract the zip folder and you're ready to go.

Source code and downloads on GitHub!
Check the Terraria Forums thread for more info.
