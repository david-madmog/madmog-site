A TERRARIA BACKUP UTILITY PROGRAM



T-Backup is a program that backs up worlds and player files from the game Terraria. It supports multiple profiles to allow different locations for mods.
It is written in C#, and is intended to be run on Windows.
T-Backup does not require an installer, simply run the downloaded executable to start.
A new, improved version is currently in the works!



You can view the source code, as well as downloads via Github!
I'd love to hear any feedback - you can do so via The Official Forum Post or you can contact me directly.

A TOTAL CONVERSION TERRARIA MOD



Super Terraria World (STW) is a standalone role-playing mod, which will be kept up-to-date with any official release of Terraria. The primary goal of this mod is to bring a MMORPG element to the game. Currently, the devs are only releasing Single Player Alpha versions of STW. This is so they can focus all of their effort on creating plenty of quality content for us to enjoy.
I developed an installer for the mod, and am currently working with the devs to create a new, more improved launcher using adobe flash builder.




You can found out more information about the mod through the forum thread, as well as joining us on discord .

A CUSTOM LOGO CREATION TOOL



Terraria logo maker is a tool that allows you to create custom logos like the one from the game Terraria. It has support for custom themes, but only includes 'Overworld' at the moment. The logo you can see here was made using this tool!
Once more themes have been made, they will be added as optional additional downloads. If you wish you can create your own theme using this tutorial.
Terraria Logo Maker is written in C# and intended to be run on windows. Simply extract the zip folder and you're ready to go.




You can view the source code, as well as downloads via Github!
I'd love to hear any feedback - you can do so via The Official Forum Post or you can contact me directly.
