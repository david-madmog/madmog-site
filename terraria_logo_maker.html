ABOUT TERRARIA LOGO MAKER

Terraria Logo Maker is a program that allows you to create a custom Terraria styled logos, with support for custom themes.
Terraria Logo Maker Currently Ships with one theme:
-Overworld
(More to come in the future)
There will also be a list of optional additional themes, once some are released.
If you wish, you can create your own themes using this tutorial! It is written in C#, and is intended to be run on Windows. To install a new theme, navigate to the folder you extracted the zip to, and drop the folder into the 'Textures' folder. (If it does not exist, create it)



To run, simply extarct the contents of the zip to a folder of your choice, then run the executable.
